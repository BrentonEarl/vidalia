Konfiguration von Vidalia und Tor

Vidalia ermöglicht es Ihnen die am gebrächlisten Einstellungen von Vidalia und Tor vorzunehmen. Außerdem können Sie mit Vidalia einen Tor-Server einrichten und betreiben und dadurch dem Tor-Netzwerk zu weiterem Wachtums verhelfen.

Allgemeine Einstellungen

Die Seite Allgemein enthält die am meisten gebrauchten Einstellungen.

  • Pfad zur Tor-Anwendung: Dies ist die ausführbare Tor-Datei, die Vidalia startet, wenn Sie Start aus dem Tray-Menü auswählen. Falls Sie mehrere Versionen von Tor installiert haben, können Sie durch Klicken auf die Wählen Schaltfläche die Tor-Version auswählen, die Vidalia benutzen soll.
  • Start Einstellungen: Durch diese Einstellung wird Tor automatisch zusammen mit Vidalia gestartet. Sie können Vidalia auch konfigurieren, dass es beim Systemstart ausgeührt wird (nur unter Windows).

Server Einstellungen

Lesen Sie diese Hilfeseite für detailierte Informationen zur Installation und zum Betrieb eines Tor-Servers.

Fortgeschrittene Einstellungen

Die Einstellungen auf der Seite Fortgeschritten sollten gewöhnlich nur von erfahrenen Benutzern vorgenommen werden.

  • Kontroll Port: Den Kontroll Port benutzt Vidalia zur Kommunikation mit Tor. Diese Einstellung braucht nur verändert zu werden, wenn Sie einen Konflikt mit einem anderen Dienst auf Ihrem System haben.
  • Tor Konfigurationsdatei: Mit dieser Option können Sie Vidalia veranlassen, Tor mit einer speziellen Konfiguarionsdatei, torrc, auszuführen.
  • Berechtigungen: Wenn Sie einen Wert für Als folgender Benutzer ausführen angeben, wird Tor mit den Rechten dieses Benutzers ausgeührt (setuid). Wenn Sie einen Wert für Als folgende Gruppe ausführen wird Tor mit den Rechten dieser Gruppe ausgeführt (setgid).
