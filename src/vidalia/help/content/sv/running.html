Användning av Tor

Med Vidalia du kan kontrolla Tor; startar och stoppar det, övervakar Tors ställning och vet om det har oväntat avslutats.

Startar och stoppar Tor

För starta Tor,

  1. Välj Starta från aktivitetsfälts Vidalias meny eller trycka Ctrl+S
    2. .
  2. Vidalia's tray icon will change from an onion with a red X to a green onion when Tor has started.
Om Vidalia kan inte starta Tor, Vidalia skall visa en felmeddelande, som berättar vad var det problem. Du kan också titta din meddelandelogg, om Tor har skrev något mer information av det problem.

För stoppa Tor,

  1. Välj Stoppa från aktivitetsfälts Vidalias meny eller trycka Ctrl+T
    2. .
  2. Vidalia's tray icon will change from a green onion to a gray onion with a red X when Tor has stopped.
Om Vidalia kan inte stoppa Tor, Vidalia skall visa en felmeddelande, som berättar vad var det problem. Du kan också titta din meddelandelogg, om Tor har skrev något mer information av det problem.

Övervakar Tors ställning

Vidalia tells you about the status of Tor by displaying an icon in your system tray or dock area. The following table shows the different states indicated by an icon in your system's notification area:

Tor har stoppat. Välj Starta från aktivitetsfälts Vidalias meny för startar Tor.
Tor startas upp. You can check the meddelandelogg for status information about Tor while it is starting.
Tor är igång. Om du vill stopppa Tor, välj Stoppa från Vidalias meny. Tor will print informational messages to the meddelandelogg while it is running, if you want to see what Tor is doing.
Tor avslutas nu.

If Tor exits unexpectedly, Vidalia will change its icon to the dark onion with a red X and display an error message letting you know what went wrong. You can also check the meddelandelogg for details about any problems Tor encountered before it exited.
