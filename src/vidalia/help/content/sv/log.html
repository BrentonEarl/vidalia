Meddelandelogg

The message log lets you see status information about a running Tor process. Each message has a severity associated with it, ranging from Fel (most serious) to Avlusa (most verbose). See the help section on message severities for more information.

Meddelandes stränghetar

A message's severity tells you how important the message is. A higher severity message usually indicates that something has gone wrong with Tor. Lower severity messages appear frequently during normal Tor operations and usually do not need to be logged.

The possible message severities, from most severe to least severe, are:

  • Fel: Messages that appear when something has gone very wrong and Tor cannot proceed. These messages will be highlighted in red in the message log.
  • Varning: Messages that only appear when something has gone wrong with Tor, but are not fatal and Tor will continue running. These messages will be highlighted in yellow.
  • Underrättelser: Messages that appear infrequently during normal Tor operation and are not considered errors, but you still may care about.
  • Info: Messages that appear frequently during normal Tor operation and are not usually of interest to most users.
  • Avlusa: Extremely verbose messages that are primarily of interest to developers. You should generally not log debug messages unless you know what you are doing.

Mest av användar skulle bara att logga Fel, Varning, och Underrättelser meddelanden.

To select which message severities you would like to see, do the following:

  1. Öppna meddelandelogg från aktivitetsfälts Vidalias meny.
  2. Klicka Inställningar från toppen av meddelandelogg.
  3. Check message severities you would like to see from the Message Filter group on the left and uncheck message severities you would like to hide.
  4. Click Spara inställningar to apply your new message filter.

Spara loggmeddelanden till en fil

Vidalia can also write log messages to a file, as well as logging them in the message log window. To enable logging to a file, follow these steps:

  1. Öppna meddelandelogg från aktivitetsfälts Vidalias meny.
  2. Klicka Inställningar från toppen av meddelandelogg.
  3. Check the box labeled Spara automatiskt nya loggmeddelanden till filen.
  4. If you would like to change the file to which messages will be written, either type the path and filename into the text box, or click Bläddra to navigate to a location for your log file.
  5. Click Spara inställningar to save your log destination.
